<h1>BLUE DARK THEME</h1>

Blue flavoured Dark Theme for Visual Studio Code.

<h3>Install</h3>
Search "blueDark Theme" in the extensions tab of your VS Code amd install.
Or get it directly from the VS Code Marketplace.
